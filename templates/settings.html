📤 Intégration — Mesures → Home Assistant
⚠️ Pourquoi l'avertissement "pas d'identifiant unique" apparaît ?
L'API POST /api/states/ de Home Assistant crée des états temporaires sans enregistrement dans le registre d'entités. Ces entités ne peuvent donc jamais avoir de unique_id, quelle que soit l'application qui les pousse. C'est une limitation de l'architecture HA, pas un bug de Pool Manager.
Solution : laisser HA tirer les données via des rest sensors qui ont un unique_id natif.
✅ Option recommandée — REST sensors HA (zéro avertissement)

HA interroge l'endpoint /api/homepage de Pool Manager toutes les 5 minutes. Les capteurs créés ont un unique_id, sont personnalisables et n'ont aucun avertissement. Aucun token HA supplémentaire requis — utilisez le token Homepage ci-dessous.

Étape 1 : Copiez ce YAML dans votre configuration.yaml Home Assistant (remplacez VOTRE_IP et VOTRE_TOKEN par le token Homepage affiché dans la section ci-dessous). 

rest:
  - resource: "http://VOTRE_IP:5000/api/homepage"
    headers:
      Authorization: "Bearer VOTRE_TOKEN_HOMEPAGE"
    scan_interval: 300
    sensor:
      - name: "Piscine pH"
        unique_id: pool_manager_ph
        value_template: "{% raw %}{{ value_json.get('ph') | float(0) }}{% endraw %}"
        state_class: measurement
        icon: mdi:water
      - name: "Piscine Brome"
        unique_id: pool_manager_bromine
        value_template: "{% raw %}{{ value_json.get('bromine') | float(0) }}{% endraw %}"
        unit_of_measurement: "ppm"
        state_class: measurement
        icon: mdi:flask
      - name: "Piscine Dureté TH"
        unique_id: pool_manager_hardness
        value_template: "{% raw %}{{ value_json.get('hardness') | float(0) }}{% endraw %}"
        unit_of_measurement: "ppm"
        state_class: measurement
        icon: mdi:water-opacity
      - name: "Piscine Alcalinité TAC"
        unique_id: pool_manager_alkalinity
        value_template: "{% raw %}{{ value_json.get('alkalinity') | float(0) }}{% endraw %}"
        unit_of_measurement: "ppm"
        state_class: measurement
        icon: mdi:flask-empty
      - name: "Piscine Température"
        unique_id: pool_manager_temperature
        value_template: "{% raw %}{{ value_json.get('temperature') | float(0) }}{% endraw %}"
        unit_of_measurement: "°C"
        device_class: temperature
        state_class: measurement
        icon: mdi:thermometer
Puis : redémarrer HA ou recharger la config YAML

Étape 2 : Une fois les REST sensors configurés dans HA, décochez "Activer l'envoi push" ci-dessous pour désactiver les entités sans unique_id.

🏠 Homepage — Widget Custom API

Intégrez votre piscine dans votre tableau de bord Homepage. Le widget affiche les valeurs en temps réel avec icônes de statut.

✓ Copié ! Token dédié à Homepage. Ne le partagez pas.
⚙️ Configuration Homepage — copiez ceci dans votre services.yaml (remplacez VOTRE_IP) : 
- Piscine:
    href: http://VOTRE_IP:5000
    description: Piscine Manager
    widget:
      type: customapi
      url: http://VOTRE_IP:5000/api/homepage
      headers:
        Authorization: Bearer {{ settings.homepage_token or 'VOTRE_TOKEN' }}
      mappings:
        - field: ph_display
          label: pH
          format: text
        - field: bromine_display
          label: Brome
          format: text
        - field: temperature
          label: Température
          format: number
          suffix: " °C"
        - field: corrections
          label: Corrections
          format: number
Champs disponibles dans l'API :
ph, bromine, hardness, alkalinity — valeurs brutes
ph_display, bromine_display, hardness_display, alkalinity_display — avec icône ✅/⬇️/⬆️
ph_status, bromine_status, … — ok / low / high / none
temperature — dernière sonde HA (°C)
last_test — date du dernier test
corrections — nombre de corrections en attente
pending_treatments — traitements en cours
📋 Valeurs de référence (piscine au brome)
Paramètre Trop bas ✅ Zone idéale Trop haut
💧 pH < 7,2 – eau corrosive 7,2 – 7,6 > 7,8 – brome inactif
🧪 Brome < 3 ppm – risque bactérien 3 – 5 ppm > 8 ppm – irritant
🪨 Dureté TH < 200 ppm – corrosion 200 – 400 ppm > 500 ppm – entartrage
⚗️ Alcalinité TAC < 80 ppm – pH instable 80 – 120 ppm > 150 ppm – pH difficile à baisser
